# Day 5 – Power BI Dashboard Development

## Objective

Designed and developed an interactive Mutual Fund Analytics Dashboard in Power BI to visualize industry trends, fund performance, investor behavior, and SIP market insights.


## Page 1 – Industry Overview

### Tasks Completed

* Created KPI cards for Total AUM, Active Folios, Average Expense Ratio, and Industry CAGR.
* Built AUM Growth Trend line chart.
* Developed Top Fund Houses by AUM visualization.
* Added category-wise distribution analysis.
* Applied dashboard branding, containers, shadows, and consistent color themes.

### Outcome

Created a high-level overview dashboard for analyzing the mutual fund industry's growth and performance.

## Page 2 – Fund Performance

### Tasks Completed

* Designed KPI cards for Alpha, Beta, Benchmark Return, and Drawdown Rank.
* Built NAV vs Benchmark comparison chart.
* Created Risk vs Return scatter plot with AUM-based bubble sizing.
* Developed sortable Fund Performance Scorecard.
* Added slicers for Fund House, Category, and Plan.

### Outcome

Enabled comparison of fund performance, risk metrics, and benchmark tracking.


## Page 3 – Investor Analytics

### Tasks Completed

* Created KPI cards for Total Investors, Transaction Amount, States Covered, and Cities Covered.
* Developed Transaction Amount by State analysis.
* Built SIP vs Lumpsum vs Redemption distribution chart.
* Created Age Group vs Average SIP Amount visualization.
* Developed Monthly Transaction Volume Trend chart.
* Added State, Age Group, and City Tier slicers.

### Outcome

Provided insights into investor demographics and transaction behavior.

## Page 4 – SIP & Market Trends

### Tasks Completed

* Created KPI cards for Active SIP Accounts, SIP Growth %, SIP Inflows, and New SIP Accounts.
* Built SIP Inflow vs NIFTY 50 dual-axis chart.
* Developed Category Inflows Heat Map using matrix visual and conditional formatting.
* Created Top Categories by Net Inflow analysis.
* Applied professional formatting and dashboard branding.

### Outcome

Visualized SIP growth trends, market performance, and category-level investment flows.

## Dashboard Enhancements

### Tasks Completed

* Added Bluestock branding and logo.
* Applied consistent dashboard theme across all pages.
* Implemented professional card layouts, shadows, borders, and containers.
* Improved visual hierarchy and dashboard readability.
* Organized pages for smooth user navigation and interaction.

## Deliverables Generated

* Power BI Dashboard (.pbix)
* Dashboard PDF Export
* Dashboard Page Screenshots
* Interactive KPI Cards
* Charts, Heat Maps, Slicers, and Visual Analytics

## Tools Used

* Power BI
* Power Query
* DAX
* Data Modeling
* Interactive Visualizations

## Day 5 Summary

Successfully completed the end-to-end Power BI dashboard consisting of four analytical pages: Industry Overview, Fund Performance, Investor Analytics, and SIP & Market Trends. Implemented KPIs, charts, heat maps, slicers, and branding elements to create a professional business intelligence dashboard for mutual fund analytics.
